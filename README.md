# GitHub CLI Practice Repository

このリポジトリは GitHub CLI (`gh`) コマンドの練習用です。

## 練習内容

- リポジトリの作成と管理
- Issue の作成と管理
- Pull Request の作成と管理
- リリースの作成と管理
- GitHub Actions の確認

## 使用するコマンド例

```bash
# リポジトリ情報の確認
gh repo view

# Issue の一覧表示
gh issue list

# PR の一覧表示
gh pr list

# リリースの一覧表示
gh release list
```
